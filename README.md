# プロジェクト名
BeaconApp

# 概要
ビーコンの発信、受信<br>
ビーコンの登録、削除（ビーコン値のQRコードの表示、読み込み）<br>
ビーコン値の受信内容の保存、参照<br>

# 開発フロー
個人開発
